# Project Name

A Python project template.

## Installation

```bash
pip install -r requirements.txt
```

```bash
python -m src.main
```

## Development

```bash
# Install development dependencies
pip install -r requirements-dev.txt

# Run tests
pytest

# Run linting
flake8 src/
black src/
```
